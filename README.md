# mongoose
Useful examples on Mongoose: Multi Insert 10.000 documents on mongodb using mongoose on nodejs

# Install
```
  apt install node npm mongo-org
  mkdir mongoose
  cd mongoose
  npm init
  npm install mongoose
```
# Use multi insert
```
  cd mongoose
  ./multi_insert_mongoose.js
```
